# Research Papers

Place the PDF files for the two research papers in this directory:

1. **Prompt Repetition** (Google, 2025) — `prompt_repetition.pdf`
   - "Prompt Repetition Improves Non-Reasoning LLMs"
   - Focus: Non-reasoning / 0-shot tasks
   - Mechanism: Exploits causal attention by duplicating the input

2. **RE2 Re-Reading** (Xu et al., EMNLP 2024) — `re2_rereading.pdf`
   - "Re-Reading Improves Reasoning in Large Language Models"
   - Focus: Reasoning tasks with Chain-of-Thought
   - Mechanism: Semantic prompt engineering

These papers are `.gitignore`-excluded by default. Add them manually.
